# 🤖 Telegram MTProto Proxy Bot with 3x-ui Sync

Асинхронный Telegram-бот на **Aiogram 3**, который автоматически синхронизируется с вашей панелью **3x-ui** через API, проверяет доступность прокси-серверов в реальном времени и выдает пользователям готовые ссылки на подключение со встроенным **спонсорским каналом** (для монетизации).

## ✨ Особенности
- **Полная автоматизация**: Больше никаких ручных текстовых файлов. Бот сам забирает свежие прокси из панели `3x-ui` по API.
- **Поддержка рекламы (Sponsor Channel)**: Бот автоматически подставляет ваш `Proxy Tag` в ссылку, закрепляя ваш канал на вершине списка чатов у пользователя.
- **Умный чекер**: Перед выдачей ссылки бот мгновенно проверяет пинг порта, чтобы исключить отправку «мертвых» прокси.
- **Два протокола**: Выдает нативную кнопку для подключения **MTProto** внутри Telegram, а также (опционально) отправляет строку **VLESS Reality** для обхода жестких блокировок всего интернета.

## 🚀 Быстрый старт

### 1. Подготовка сервера (VPS)
1. Установите панель управления **3x-ui** на ваш сервер (Ubuntu/Debian):
   ```bash
   bash <(curl -Ls https://githubusercontent.com)
   ```
2. Создайте в панели подключение с протоколом `mtproto` (и `vless` по желанию).
3. Зарегистрируйте ваш прокси в официальном боте `@MTProxybot` и получите рекламный **Proxy Tag**. Не забудьте указать этот тег в настройках подключения в панели 3x-ui.

### 2. Установка и запуск бота

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com
   cd ИМЯ_РЕПОЗИТОРИЯ
   ```

2. Установите необходимые зависимости:
   ```bash
   pip install aiogram aiohttp
   ```

3. Откройте файл `bot.py` и заполните конфигурационные данные:
   ```python
   BOT_TOKEN = "ВАШ_ТОКЕН_ОТ_BOT_FATHER"
   SPONSOR_TAG = "ВАШ_ПРОКСИ_ТЕГ_ИЗ_MTPROXYBOT"

   PANEL_URL = "http://IP_ВАШЕГО_СЕРВЕРА:ПОРТ"
   PANEL_LOGIN = "ЛОГИН_ОТ_ПАНЕЛИ"
   PANEL_PASSWORD = "ПАРОЛЬ_ОТ_ПАНЕЛИ"
   ```

4. Запустите бота:
   ```bash
   python bot.py
   ```

## 🛠 Технологический стек
- **Язык**: Python 3.10+
- **Фреймворк**: Aiogram 3.x (Асинхронный)
